# Linux System Analyzer

A simple Python Tkinter application for checking basic Linux system information.

## Features

- Shows current working directory
- Lists files in the current directory
- Finds `.txt` files
- Shows CPU and memory usage processes
- Uses a Tkinter GUI with Listbox and Scrollbar

## Technologies

- Python
- Tkinter
- os
- pathlib
- subprocess

## How to run

```bash
python3 main.py
```

## Note

This project is designed mainly for Linux systems.
